Consulta 1

1. Obtener la marca, modelo y precio de los vehículos que cuesten igual o mas de 87000 €.

for $nombre in doc("deportes.xml")/equipos_de_futbol/equipo/nombre/text() return {$nombre}

Consulta 2

2. Muestra los jugadores cuya edad sea mayor de 30 años.

for $jugador in doc("deportes.xml")/equipos_de_futbol/equipo/jugadores/jugador where $jugador/edad >= 30 return $jugador

Consulta 3

3. Mostrar sólo el nombre sin atributos de los jugadores cuya edad sea menor o igual a 30.

for $jugador in doc("deportes.xml")/equipos_de_futbol/equipo/jugadores/jugador[edad<=30] return {$jugador/nombre/text()}

Consulta 4

4. Mostrar los jugadores ordenados primero por "posicion" y luego por "jugador" en una sola consulta.

for $jugador in doc("deportes.xml")/equipos_de_futbol/equipo/jugadores/jugador order by $jugador/@posicion,$jugador/nombre return $jugador

Consulta 5

5. Mostrar el año mínimo y máximo de los jugadores

let $max := max(doc("deportes.xml")/equipos_de_futbol/equipo/jugadores/jugador/edad), $min := min(doc("deportes.xml")/equipos_de_futbol/equipo/jugadores/jugador/edad) return {$max} {$min}

Consulta 6

6. Mostrar los jugadores que terminan por la letra "o".

for $jugador in doc("deportes.xml")/equipos_de_futbol/equipo/jugadores/jugador where ends-with($jugador/nombre, "o") return $jugador

Consulta 7

7. Mostrar los equipos que tengan una "S" mayúscula o minúscula en el nombre.

for $equipo in doc("deportes.xml")/equipos_de_futbol/equipo where contains($equipo/nombre, "s") or contains($equipo/nombre, "S") return $equipo

Consulta 8

8. Mustra los jugadores que sean de nacionalidad Española

for $jugador in doc("deportes.xml")/equipos_de_futbol/equipo/jugadores/jugador[nacionalidad="española"] return {$jugador/nombre/text()}

Consulta 9

9. Mustra el nombre del equipo y el numero de caracteres que tiene.

for $equipo in doc("deportes.xml")/equipos_de_futbol/equipo return {$equipo/nombre} {string-length($equipo/nombre)}

Consulta 10

10. Mostrar los nombres de los equipos en una tabla de HTML.

{ for $equipo in doc("deportes.xml")/equipos_de_futbol/equipo return }
{$equipo/nombre/text()}
